\emph{
	Security monitoring systems based on the Internet of Things (IoT) require reliable data communication, real-time device health monitoring, and efficient server infrastructure management. This study extends and optimizes the existing MIOTVision CCTV monitoring platform based on ESP32-CAM, which integrates a web-based monitoring platform, a mobile application, and camera infrastructure. The focus of this study includes the implementation of intruder and motion detection functionalities based on frame comparison, and a comprehensive device telemetry monitoring system (RSSI, uptime, free heap, reset reason, capture/publish success rate) transmitted in real-time. To support stable real-time communication, MQTT over WSS is stabilized and Laravel Reverb is integrated. Furthermore, operational reliability is enhanced by introducing Over-the-Air (OTA) firmware update support, a MySQL database retention mechanism, and automatic MinIO object storage cleanup. The infrastructure is deployed using Cloudflared Tunnel for secure public access and Webmin to improve the maintainability of the Intel NUC server. The evaluation results demonstrate that these infrastructure optimizations and new feature implementations successfully improve performance, storage efficiency, data communication stability, and overall server maintainability.}
