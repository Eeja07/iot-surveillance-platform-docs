Sistem pemantauan keamanan berbasis Internet of Things (IoT) memerlukan keandalan komunikasi data, pemantauan kondisi perangkat secara real-time, serta efisiensi pengelolaan infrastruktur server. Penelitian ini memperluas dan mengoptimalkan platform monitoring CCTV MIOTVision berbasis ESP32-CAM yang mengintegrasikan website monitoring, aplikasi mobile, dan infrastruktur kamera. Fokus penelitian ini adalah mengimplementasikan fitur deteksi penyusup (intruder detection) dan deteksi gerakan (motion detection) berbasis perbandingan frame citra, serta sistem pemantauan telemetri perangkat (RSSI, uptime, free heap, reset reason, capture/publish success rate) yang dikirimkan secara real-time. Untuk mendukung komunikasi real-time yang stabil, dilakukan stabilisasi protokol MQTT over WSS dan integrasi Laravel Reverb. Selain itu, keandalan operasional ditingkatkan melalui penambahan fitur pembaruan firmware secara nirkabel (OTA), mekanisme retensi basis data MySQL, dan pembersihan otomatis objek penyimpanan MinIO (object cleanup). Infrastruktur dideploy menggunakan Cloudflared Tunnel untuk akses publik yang aman dan Webmin untuk kemudahan administrasi server Intel NUC. Hasil pengujian menunjukkan bahwa optimalisasi infrastruktur dan penambahan fitur-fitur baru ini berhasil meningkatkan performa, efisiensi penyimpanan, stabilitas komunikasi data, serta kemudahan pemeliharaan server secara keseluruhan.
